## YTDLP-GUI
A graphical shell using the yt-dlp library. Built with Python and PySide6.

---

### Installation & Running

#### For Windows users
* **[Download Latest Version for Windows](https://github.com/r3cruit1337/ytdlp-gui/releases/latest/download/ytdlp-gui.exe)** 🚀
* Just download the `ytdlp-gui.exe` and run it (no installation required).

#### For Linux users (Run from source)
You can easily run the application directly using your system's Python:

Clone this repository or download the ZIP archive:
   git clone this repo into the some foler or directory
   open some IDE or code editor and cd ytdlp-gui

Install the required dependencies:
    pip install -r requirements.txt

Launch the application:
    python src/main.py
